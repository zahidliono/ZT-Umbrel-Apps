⛏️ Connect Your Miner

Point your SHA-256 ASIC miner at this node using its RPC endpoint. Replace YOUR_FB_ADDR with your Fractal Bitcoin address from UniSat Wallet. Your RPC password is the same as your Umbrel login password.

cpuminer / minerd
./minerd \
  -o http://umbrel.local:10332 \
  -u fractal -p YOUR_UMBREL_PASSWORD \
  --coinbase-addr=YOUR_FB_ADDR \
  -a sha256d --no-longpoll
CGMiner / BFGMiner
cgminer \
  -o http://umbrel.local:10332 \
  -u fractal -p YOUR_UMBREL_PASSWORD \
  --coinbase-addr=YOUR_FB_ADDR

🔧 RPC Commands (via SSH)

docker exec fractal-store-solo-node_node_1 \
  bitcoin-cli -datadir=/data getmininginfo
