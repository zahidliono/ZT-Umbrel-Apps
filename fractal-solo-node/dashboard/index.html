⛏️ Connect Your Miner

Point your SHA-256 ASIC miner (or cpuminer / BFGMiner) at this node's RPC endpoint. Replace YOUR_FB_ADDRESS with your Fractal Bitcoin wallet address (use UniSat Wallet).

Miner connection (cpuminer example):
./minerd -o http://umbrel.local:10332 \ -u fractal -p fractal_umbrel_node \ --coinbase-addr=YOUR_FB_ADDRESS \ -a sha256d --no-longpoll
CGMiner / BFGMiner:
cgminer -o http://umbrel.local:10332 \ -u fractal -p fractal_umbrel_node \ --coinbase-addr=YOUR_FB_ADDRESS
For pool software (ViaBTC / CKPool) — RPC config:
{ "host": "umbrel.local", "port": 10332, "user": "fractal", "pass": "fractal_umbrel_node" }
Merged mining ChainID (AuxPoW):
ChainID: 0x2024 (decimal: 8228)
⚠️ Solo mining without significant SHA-256 hashpower (ASICs) is statistically unlikely to find blocks. Permissionless difficulty is ~5 GH and merged-mining difficulty is ~400 GH. For small miners, consider using an established Fractal Bitcoin pool and just running this node for sovereignty.
